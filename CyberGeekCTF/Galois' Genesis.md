# Galois' Genesis

## Challenge Overview
- Event: CyberGeek CTF
- Category: Crypto
- Prompt: Decrypt the ciphertext `0eRbrk{_ng}eoe3Z|aEm1g` using the `Galois` clue.
- Final Flag: geek{Z0mb1E_g3nera|oR}

## Executive Summary
The ciphertext length and challenge title point to GF(23). Treating character positions as the nonzero field elements and inverting the primitive-root permutation recovers the flag cleanly.

## Solution Walkthrough
### Observation
The ciphertext has length 22:

```text
0eRbrk{_ng}eoe3Z|aEm1g
```

`22 = 23 - 1`, which matches the number of nonzero elements in the finite field
`GF(23)`. The nonzero elements of `GF(23)` form a cyclic multiplicative group,
so a primitive root modulo 23 can generate all 22 positions.

Primitive roots modulo 23 are:

```text
[5, 7, 10, 11, 14, 15, 17, 19, 20, 21]
```

Testing these roots against the known flag prefix `geek{` identifies the
permutation generated by `7` modulo 23.

### Decryption
Generate the nonzero field elements as powers of 7 modulo 23, then rotate the
sequence once so it starts at 7:

```text
[7, 3, 21, 9, 17, 4, 5, 12, 15, 13, 22, 16, 20, 2, 14, 6, 19, 18, 11, 8, 10, 1]
```

Convert each residue to a zero-based index by subtracting 1:

```text
[6, 2, 20, 8, 16, 3, 4, 11, 14, 12, 21, 15, 19, 1, 13, 5, 18, 17, 10, 7, 9, 0]
```

The ciphertext was made by reading the plaintext at those indices. To decrypt,
place each ciphertext character back into its generated index.

### Reproduction Script
```python
ct = "0eRbrk{_ng}eoe3Z|aEm1g"
N = len(ct)
p = 23
g = 7

seq = []
x = 1
for _ in range(N):
    seq.append(x)
    x = (x * g) % p

seq = seq[1:] + seq[:1]
indices = [r - 1 for r in seq]

flag = ["?"] * N
for i, ch in enumerate(ct):
    flag[indices[i]] = ch

print("".join(flag))
```

Output:

```text
geek{Z0mb1E_g3nera|oR}
```

### Recheck
Re-encrypting the recovered flag with the same index list gives the original
ciphertext exactly:

```python
flag = "geek{Z0mb1E_g3nera|oR}"
indices = [6, 2, 20, 8, 16, 3, 4, 11, 14, 12, 21, 15, 19, 1, 13, 5, 18, 17, 10, 7, 9, 0]

print("".join(flag[i] for i in indices))
```

Output:

```text
0eRbrk{_ng}eoe3Z|aEm1g
```

The brute-force recheck over all primitive roots and rotations found the same
permutation twice: once from root `7`, and once from root `10` reversed. This is
expected because `10` is the modular inverse of `7` modulo 23.
